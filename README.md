# stacks-core

Stacks Core contains foundational components and shared logic used across
Stacks-based applications.

This repository focuses on reusable primitives, core contracts, and common
patterns that can be leveraged by multiple projects.

## Scope
- Core Clarity contracts
- Shared utilities and helpers
- Common architectural patterns
- Protocol-level building blocks

## Use Cases
- Base contracts for other projects
- Shared libraries
- Internal dependencies

Actively maintained.
